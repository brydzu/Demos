CSS Grid Paradise Puzzle

Playing the game

With the Windows Fall Creator's Update we will be shipping an updated implementation of CSS Grid.

The game consists of a 3 by 3 grid with a shuffled set of images. The goal of the game is to move one of the images that is next to the open space into the open space. Continue this process until the order of the images is sequential. If you're doing this game audibly, you will hear the following completed quote by Scott Hamilton.

  • Row 1: The only disability
  • Row 2: in life is
  • Row 3: a bad attitude

Helpful keyboard shortcuts:

C
This will audibly outline the current state of the game telling you which pieces are in each row
H
This will audibly tell you in which row and column the open space is
M
This will audibly tell you which pieces are possible to move into the open space
Arrow Keys
This will move the piece that is able to move in that direction. For example, if you press the up arrow key the piece that is directly below the open space will move up.
Numeric keys 1 through 9
This will try to move the image with that index
